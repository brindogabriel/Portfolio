FomentAR

Es un sistema de autogestion de sociedades de fomento

Problema

La sociedad de fomento contaba con los registros en formato papel y a veces se dificultaba o se perdia algun papel

Solución

Se creó un sistema de gestion que cuenta con una interfaz web y una base de datos

Tecnologias utilizadas

PHP MYSQL BOOTSTRAP JAVASCRIPT HTML CSS
