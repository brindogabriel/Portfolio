FomentAR

FomentAR es un proyecto que busca solucionar el problema de la gestión de sociedades de fomento que aún tienen sus registros en formato papel. Esto puede dificultar el acceso a cierta información o incluso hacer que se pierda algún documento

Solución
Para solucionar este problema, se ha creado un sistema de gestión que cuenta con una interfaz web y una base de datos. Esto permite que la información esté siempre disponible y sea más fácil de acceder.
Tecnologías utilizadas
PHP MYSQL BOOTSTRAP JAVASCRIPT HTML GIT CSS SELECT2 DATATABLES FULLCALENDAR

Estas herramientas han permitido desarrollar la interfaz web y la base de datos del sistema, así como añadir diferentes funcionalidades y hacer que el sistema sea intuitivo y fácil de usar.
